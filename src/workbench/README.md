# Workbench

Installs these pinned commands on the system-wide PATH of Debian- and
Ubuntu-based Dev Containers on amd64 and arm64:

| Command | Version |
| --- | --- |
| `codex` | 0.144.3 |
| `claude` | 2.1.197 |
| `pi` | 0.80.6 |
| `wt` | 0.67.0 |
| `glow` | 2.1.2 |
| `fd` | 10.4.2 |
| `rg` | 15.1.0 |

The Feature installs no credentials, shell integration, or agent configuration.
It depends on the official `common-utils` and `node` Features.
