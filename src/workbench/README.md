# Workbench

Installs a one-stop command-line baseline on the system-wide PATH of Debian-
and Ubuntu-based Dev Containers on amd64 and arm64:

| Command | Version |
| --- | --- |
| `codex` | 0.144.3 |
| `claude` | 2.1.197 |
| `pi` | 0.80.6 |
| `hunk` | 0.17.0 |
| `wt` | 0.67.0 |
| `glow` | 2.1.2 |
| `fd` | 10.4.2 |
| `rg` | 15.1.0 |
| `lazygit` | 0.63.0 |
| `jq` | Distribution package (from `common-utils`) |
| `bat` | 0.26.1 |
| `delta` | 0.19.2 |
| `herdr` | 0.7.3 |
| `nvim` | 0.12.4 |
| `zsh` | Distribution package |
| `gh` | 2.96.0 |
| `fzf` | 0.74.0 |
| `zoxide` | 0.10.0 |
| `starship` | 1.26.0 |

The Feature installs no credentials, shell initialization, login-shell changes,
or agent configuration. Personal configuration remains the responsibility of
your dotfiles. It depends on the official `common-utils` and `node` Features.
